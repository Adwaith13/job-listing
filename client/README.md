# JobListing Site

Welcome to the JobListing site! This platform is designed for recruiters to easily post and manage job listings.

## Features

- **User Authentication:** Recruiters need to log in to access the functionality to post and edit job listings.
- **Post Jobs:** Recruiters can post new job listings by providing job details such as title, description, requirements, etc.
- **Edit Jobs:** Recruiters have the ability to edit existing job listings to keep them up-to-date.
- **View Jobs:** Job seekers and recruiters can view the list of available jobs.

## Getting Started

Follow these steps to set up and run the JobListing site on your local machine.

### Prerequisites

- Node.js: [Download and Install Node.js](https://nodejs.org/)
- MongoDB: [Download and Install MongoDB](https://www.mongodb.com/try/download/community)

## Technologies Used

- Node.js
- Express.js
- MongoDB
- React.js
- JWT (for authentication)
